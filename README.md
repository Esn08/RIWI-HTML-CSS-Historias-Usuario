## Riwi Module II: User story about personal portfolio using HTML, CSS, and JavaScript

Welcome, this is the user story about a personal portfolio repository. This project is a responsive website 
designed to showcase software development skills, featured projects, and professional background using HTML5, CSS y JavaScript.

## Features

- **Responsive Design:** Adapted for Desktop, Tablet, and Mobile screens using CSS Grid and Flexbox.
- **Navigation:** Smooth scrolling navigation to different sections (About Me, Projects, Contact) and a dedicated page for Pets (*Mascotas*).
- **Hero Section:** Includes a dynamic introduction and an interactive "Change Colors" feature.
- **Projects Gallery:** A categorized showcase of my work, including:
    - **Coffee Blend:** A vintage-style coffee shop website.
    - **Euphony Pro:** A music industry event management site.
    - **Course:** An educational platform for online courses.
    - **Cakeshop:** A luxury bakery website.
- **Interactive UI:**    - Reveal/Hide text functionality in the "About Me" section.
- **Contact Form:** A layout for users to get in touch via email or phone.

## Technologies Used

- **HTML5:** Semantic structure of the application.
- **CSS3:** Styling, custom layouts, and responsive design (`styles.css`, `pets-styles.css`).
- **JavaScript:** Interactivity, DOM manipulation, and event handling (`script.js`).

##  How to Run

1. **Clone the repository:**   git clone https://github.com/Esn08/RIWI-HTML-CSS-Historias-Usuario.git
2. **Navigate to the project folder:**  User Store folder   
3. **Open the project:**   open the `index.html` file in your preferred web browser to view the site.

## Project Structure


- **assets**          # Images and icons (Logo, Hero image, Project screenshots)
- **index.html**       # Main entry point (Landing page)
- **mascotas.html**    # Secondary page for Pets
- **styles.css**       # Main stylesheet
- **pets-styles.css**  # Stylesheet for Pets page
- **script.js**        # Logic for interactivity
